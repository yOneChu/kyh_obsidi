도면에서 **PITCH**처럼 자주 나오고 꼭 알아야 하는 용어를 정리하면 아래 정도는 기본으로 보면 돼. 특히 기계/엘리베이터 부품 도면 볼 때 자주 나와.

## 1. 치수 관련 용어

| 용어                  | 의미          | 쉽게 말하면                             |
| ------------------- | ----------- | ---------------------------------- |
| **PITCH**           | 반복되는 형상 간격  | 구멍 중심 간 거리, 볼트 간 거리                |
| **DIMENSION**       | 치수          | 길이, 높이, 폭 등 숫자로 표시된 값              |
| **TOLERANCE**       | 공차          | 허용 오차. 예: 100±0.5면 99.5~100.5 가능   |
| **REF / REFERENCE** | 참고치수        | 제작 기준 치수는 아니고 참고용 치수               |
| **TYP / TYPICAL**   | 동일 조건 반복    | 같은 형상에 동일 치수 적용                    |
| **QTY**             | 수량          | QTY 4 = 4개                         |
| **THK / T**         | 두께          | THK 6 = 두께 6mm                     |
| **L / W / H**       | 길이 / 폭 / 높이 | Length / Width / Height            |
| **ID / OD**         | 내경 / 외경     | Inside Diameter / Outside Diameter |

---

## 2. 구멍/가공 관련 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**Ø / DIA**|직경|Ø10 = 지름 10mm|
|**HOLE**|구멍|일반 구멍|
|**TAP**|탭 가공|나사산이 있는 구멍|
|**DRILL**|드릴 가공|단순 구멍 뚫기|
|**C'BORE / COUNTERBORE**|카운터보어|볼트 머리가 들어가도록 계단 가공|
|**C'SINK / COUNTERSINK**|카운터싱크|접시머리 볼트용 경사 가공|
|**SLOT**|장공|길쭉한 구멍|
|**PCD**|Pitch Circle Diameter|원형 배치된 구멍들의 중심 원 지름|
|**CLEARANCE HOLE**|여유 구멍|볼트가 그냥 통과하는 구멍|

예를 들어 `4-Ø12 HOLES PCD 100` 이면  
**지름 12 구멍 4개가 지름 100짜리 원 위에 배치**되어 있다는 뜻이야.

---

## 3. 형상/모서리 관련 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**R / RADIUS**|반지름|둥근 모서리 반경|
|**FILLET**|필렛|안쪽 또는 바깥쪽 둥근 처리|
|**CHAMFER / C**|모따기|날카로운 모서리를 비스듬히 깎음|
|**C1, C2**|모따기 크기|C1 = 1mm 모따기|
|**BEVEL**|경사 가공|비스듬하게 깎은 면|
|**EDGE**|모서리|부품의 끝단|
|**CORNER**|코너|모서리 부분|

예를 들어 `C2`는 보통 **모서리를 2mm 정도 45도 방향으로 깎는다**는 의미로 많이 사용해.

## 4. 기준선/중심/위치 관련 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**CENTER LINE / CL**|중심선|구멍, 원, 대칭의 중심 기준선|
|**DATUM**|기준면/기준점|치수나 형상 측정의 기준|
|**OFFSET**|편심/이격|기준선에서 떨어진 거리|
|**SYMMETRY**|대칭|좌우 또는 상하가 같은 형상|
|**ALIGN**|정렬|중심이나 면을 맞춤|
|**BASE LINE**|기준선|치수 산출 기준이 되는 선|

도면에서 가장 중요한 건 **어디를 기준으로 치수가 잡혀 있는지**야.  
기준면이나 중심선을 잘못 보면 가공 위치가 틀어질 수 있어.

---

## 5. 도면 보기 방식 관련 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**FRONT VIEW**|정면도|앞에서 본 모습|
|**TOP VIEW**|평면도|위에서 본 모습|
|**SIDE VIEW**|측면도|옆에서 본 모습|
|**SECTION / SEC.**|단면도|잘라서 내부를 보여주는 그림|
|**DETAIL VIEW**|상세도|특정 부위를 확대해서 보여줌|
|**SCALE**|축척|실제 크기 대비 도면 크기|
|**NTS**|Not To Scale|축척 없음, 치수값을 우선 봐야 함|

특히 `SECTION A-A`는 **A-A 선을 따라 잘랐을 때 보이는 내부 형상**이라는 뜻이야.

## 6. 재질/표면/처리 관련 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**MATERIAL / MAT'L**|재질|철판, 알루미늄, SUS 등|
|**SURFACE ROUGHNESS**|표면거칠기|표면을 얼마나 매끄럽게 가공할지|
|**FINISH**|표면처리|도장, 도금, 연마 등|
|**PLATING**|도금|아연도금 등|
|**PAINT**|도장|페인트 처리|
|**HEAT TREATMENT**|열처리|강도나 경도 확보를 위한 처리|
|**BURR REMOVE**|버 제거|가공 후 날카로운 찌꺼기 제거|

`REMOVE BURR` 또는 `DEBURR`가 있으면 **가공 후 날카로운 부분을 제거하라**는 뜻이야.

## 7. 판금/용접 도면에서 자주 나오는 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**BEND**|절곡|판을 접는 가공|
|**BENDING LINE**|절곡선|접히는 기준선|
|**FLANGE**|플랜지/접힌 날개부|판금에서 꺾여 나온 부분|
|**WELD**|용접|부품을 붙이는 작업|
|**SPOT WELD**|점용접|점 형태로 용접|
|**BUTT WELD**|맞대기 용접|두 부품 끝을 맞대어 용접|
|**FILLET WELD**|필릿 용접|ㄱ자 형태 접합부 용접|

판금 도면에서는 **THK, BEND, R, FLANGE, BURR REMOVE**를 특히 자주 봐.

## 8. 조립/방향 관련 용어

|용어|의미|쉽게 말하면|
|---|---|---|
|**ASSY / ASSEMBLY**|조립품|여러 부품이 조립된 상태|
|**PART**|단품|개별 부품|
|**LH / RH**|좌측 / 우측|Left Hand / Right Hand|
|**UP / DOWN**|위 / 아래|설치 방향|
|**FR / RR**|전면 / 후면|Front / Rear|
|**MIRROR**|대칭품|좌우 반대 형상|
|**EXPLODED VIEW**|분해도|조립 관계를 펼쳐서 보여준 그림|

엘리베이터 쪽 도면에서는 **LH/RH 방향**, **FRONT/REAR**, **UP/DOWN**을 잘못 보면 현장 설치 방향이 틀어질 수 있어.

---

## 9. 도면에서 특히 조심해야 하는 표현

|표현|주의할 점|
|---|---|
|**PITCH**|끝단 간격이 아니라 보통 중심 간 거리|
|**REF**|실제 제작 기준 치수가 아닐 수 있음|
|**TYP**|같은 형상 전체에 반복 적용|
|**NTS**|그림 비율 믿지 말고 숫자 치수를 봐야 함|
|**± 공차**|허용 범위 안에 들어와야 함|
|**PCD**|원 위에 배치된 구멍 중심 기준|
|**SECTION**|실제 외형이 아니라 절단해서 본 내부 형상|

핵심만 외우면 우선 이 정도야.

**도면 볼 때 우선순위는**  
`재질 → 두께/크기 → 구멍 위치 → PITCH/PCD → 공차 → 표면처리 → 조립 방향`  
순서로 보면 실수가 줄어들어.
