**| K도면**
- K도면은 본사 도면을 이용해서 법인화 시킨 도면이에요.
- 본사 도면은 한글이지만 K 도면은 중국어 본사 도면은 KS 규격이지만 법인 도면은 BS 규격.
- C도면은 본사에서 채번할 때 앞에 C를 붙임

---
PLAN-C 또는 SUB SYSTEM이라고 부르고 둘다 같은 의미에요


SUB SYSTEM 순서는 다음과 같아요
1.  본사/법인의 각 연구소 부품 개발 담당자들이 본사/법인용 통합 도면 작성 (도면에 한글, 영어, 중국어 모두 작성)
2. 도면 작업이 끝나면 본사에서 도면번호 생성하고 PLM에 도면 등록해요 (도면은 같이 작성하지만 본사에서 관리)
3. 완료된 도면으로 자재 생성 (공급 유형에 따라서 자재번호 다름/ 13자리 또는 14자리)
4. 공급 유형에 따른 자재 채번  
    **A. 법인 경유 :** 
	- 본사 → 법인 발주 / 법인에서 중국 현지 협력사에서 물건 받고 패킹해서 본사로 납품 
	- (`14자리 자재번호`)  
	- `본사 -> 법인 -> 법인협력업체`
    **B. 직도입 (직수입)**
	 - 본사 → 중국 현지 협력사에 직접 발주 /  협력사에서 패킹해서 본사로 납품 
	 - (`13자리 자재번호`)  
	 - `본사 -> 법인협력체`
    **C. 국내 핸들링 업체**
	 -  본사 → 국내 핸들링 업체 발주 / 국내 핸들링 업체가 중국 현지 협력사에서 물건 받고 패킹해서 본사로 납품  (13자리 자재번호)
	 - `본사 -> 국내업체 -> 법인협력업체`

PLAN-C의 경우 일반 호기 (LUXEN, NEX) 또는 글로벌 제품 (LUXEN-G, NEX-G)에 모두 도입 예정이에요
