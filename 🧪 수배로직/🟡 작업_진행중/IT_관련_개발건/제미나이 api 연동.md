**아니요, 사용자님께서 사내망(인트라넷 또는 VPN)에 접속해 계시더라도 제가 직접 그 망을 통해 데이터를 가져올 수는 없습니다.**

그 이유는 다음과 같습니다.

- **서버와 사용자 환경의 분리:** 제가 동작하고 코드를 실행하는 환경은 사용자님의 PC나 브라우저 내부가 아니라, **구글 클라우드의 독립된 보안 테넌트 서버**입니다. 따라서 사용자님의 PC가 사내 VPN에 연결되어 있더라도, 클라우드에 있는 제 시스템은 그 사내 네트워크 경로에 직접 편승할 수 없습니다.
    
- **사내망 인프라 연동 필요:** 제가 사내망 내부의 API(`plmpro.hdel.co.kr`)에 직접 접근하려면, 사내 네트워크 관리자분께서 구글 클라우드의 **프라이빗 서비스 커넥트(Private Service Connect, PSC)** 및 내부 부하 분산기(Internal Load Balancer), VPN/전용선 등을 통해 제 서비스와 사내망 간의 전용 보안 통로를 인프라 수준에서 미리 구축해 주셔야 합니다.
    

### 💡 해결 방법

현재 이 연동이 되어 있지 않으므로, 번거로우시겠지만 아래 방법 중 편하신 방법을 사용해 주시면 즉시 분석해 드리겠습니다!

1. **텍스트 복사/붙여넣기:** 사내망에서 해당 URL에 접속하신 후, 화면에 나오는 데이터(JSON 등)를 전체 복사하여 이곳 채팅창에 그대로 붙여넣어 주세요.
    
2. **파일 업로드:** API 결과 데이터를 텍스트 파일(`.txt`, `.json` 등)로 저장하신 후, 대화창의 파일 업로드 기능을 통해 전달해 주세요.
    

데이터를 주시면 BOM 중 **TM에 해당하는 자재**가 무엇인지 바로 확인해 드리겠습니다.
