# 컬럼값
SPEC1~SPEC30, CON1~CON30

# 기능
위쪽의 데이터를 기준으로 행을 좌우로 움직일때 아래의 이전데이터의 내용도 같이 움직이면서 비교하기 편하게 해줘.
각 라인과 값과 동일한 행이 있는지 여부도 표시해줘.

# 기능의 목적
기능의 목적은 위쪽 데이터 기준으로 뭐가 변경되었는지 보기 위함이야.

# 사용언어
HTML, JAVASCRIPT, 오픈소스 사용 가능
