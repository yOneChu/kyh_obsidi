| BLOCK SCOPE 관리
각 담당자마다 BLOCK SCOPE 관리를 해줘야 한다. 엑셀파일(PLM > 문서)도 수정 필수 
(CAL_BOM_SUPPLIER_DESIGN)



내가 최근, 이전 데이터의 값들을 라인별로 컬럼값 비교기능을 개발하려고해.
위쪽은 최근데이터, 아래쪽은 이전데이터로 보여줘서 비교하는 기능을 만들어줘.

컬럼값
SPEC1, CON1, SPEC2, CON2, SPEC3, CON3, SPEC4,CON4, SPEC5,CON5

기능
최근데이터를 기준으로 행을 움직일때 이전데이터의 내용도 같이 움직이면서 비교하기 편하게 해줘.
각 라인과 값과 동일한 행이 있는지 여부도 표시해줘.

사용언어
HTML, JAVASCRIPT
