| B181A03 | CWT ROPE END |
| ------- | ------------ |
| B121B01 | ROPE END     |


**| 1:1 로핑**
- 1:1로핑일 경우에 탑빔에 샤클을 체결하는 용도이다.
	- 1:1로핑일 경우에는 샤클이 TOP BEAM에 체결된다. 즉, ROPE END와 TOP BEAM 도면봐서 조립해보면 딱 맞는다.
	- CWT경우 샤클이 CWT FRAME에 체결된다. 이거도 도면 두개 보면 딱 맞는다.

**| 1:1 로핑 외**
- 1:1로핑이 아닐 경우에는 기계대에 히치플레이트에 체결하기 때문에, ROPE END가 없다.
