**회의록 요약**

이번 회의의 핵심은 `BUFFER OIL` 자재가 `L 단위`로 수배되어야 하는데, 일부 로직/자재 기준에서 `BOX 단위`로 연결되면서 원가가 과대 반영될 수 있는 문제가 발생했다는 내용입니다. 자동전사 오인식이 일부 있어, 의미가 명확한 부분 위주로 정리했습니다.

**주요 이슈**

| 구분    | 내용                                                             |
| ----- | -------------------------------------------------------------- |
| 문제    | BUFFER OIL 자재 단위가 `L`와 `BOX` 기준으로 혼재되어 있음                      |
| 원인    | BOM/로직/자재마스터/구매 단위에 대한 기준이 명확히 공유되지 않음                         |
| 영향    | 실제 필요한 수량은 `3.6L`, `0.9L` 수준인데 BOX 단위로 인식되면 원가가 크게 증가해 보일 수 있음 |
| 예시    | L당 약 3,000원 수준이어야 하는데 BOX 단가 약 92,000원 기준으로 반영되는 상황 발생         |
| 발견 경로 | 원가/견적 검토 과정에서 문제 발견                                            |
| 현재 상태 | 물류/실물 지급 자체보다는 원가 및 수배 기준상 오류 가능성이 핵심                          |

**회의에서 정리된 방향**

1. BUFFER OIL 관련 자재는 우선 기존 현행 기준대로 `L 단위`로 정리한다.
    
2. 문제가 된 로직의 자재번호를 BOX 기준 자재에서 L 기준 자재로 변경한다.
    
3. 기존 설계/기수배 현장도 동일하게 조치가 필요한지 확인한다.
    
4. 구매, 조달, 설계, 마스터 관리 담당 간에 공통자재 기준을 상호 확인한다.
    
5. 단순히 이번 건만 수정하지 말고, 공통자재 중 유사하게 이중 관리되는 자재가 있는지 조사한다.
    

**추가 논의 내용**

- 출하 쪽은 실제 사용 수량을 보고 `4L 단위` 제품을 출하할 수 있음.
    
- 구매 쪽은 실물 구매 단위가 `4L`, `BOX`, `드럼` 등으로 운영될 수 있어 설계 BOM 수량과 구매 발주 단위가 다를 수 있음.
    
- 실제 업체 공급 단위가 `4L`만 가능한지, `1L` 소분 공급이 가능한지 추가 확인 필요.
    
- 1L 소분 공급은 가능하더라도 별도 소분 업체를 거쳐야 해 비용 상승 가능성이 있음.
    
- 따라서 단기적으로는 L 단위 기준을 유지하고, 장기적으로는 실제 구매/출하 단위 기준을 명확히 재정비하기로 함.
    

**조치사항**

|담당/대상|조치 내용|
|---|---|
|로직 담당|BUFFER OIL 관련 로직의 자재번호를 L 단위 자재 기준으로 변경|
|설계/수배 담당|기설계 현장 중 오류 반영 가능성이 있는 현장 확인 및 조치|
|구매/조달 담당|BUFFER OIL의 실제 공급 단위 확인: 4L, 1L, 드럼, BOX 등|
|관련 담당자|업체가 실제로 어떤 단위로 공급 가능한지 확인 후 회신|
|공통자재 담당|공통자재 중 BOM/구매/출하 단위가 이중 관리되는 항목 조사|
|전체|향후 공통자재 등록/변경 시 설계, 구매, 마스터 담당 간 트리플 체크 수행|

**결론**

이번 건은 단순 자재번호 오류라기보다, `설계 BOM 수량`, `구매 단위`, `출하 단위`, `자재마스터 기준`이 서로 다르게 관리되면서 발생한 문제입니다. 우선 긴급 조치로 BUFFER OIL 로직은 `L 단위 자재` 기준으로 수정하고, 이후 유사 공통자재까지 확대 점검하는 방향으로 정리되었습니다.
